# Gezi-Yorum
Android için gezi takip ve yönetimi uygulaması

*Konum takibi
*Bireysel gezi
*Takım gezisi
*Takım üyeleri arasında konum takibi
*Gezi paylaşma
*Sosyal medya ortamı
*Gezi indirme
*İndirilen geziyi rota olarak belirleme
